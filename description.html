Introduction

The OCR Image Translator sample combines the Bing OCR Control and the Bing
Translator Control into a single application that takes a picture of printed
text, reads the text, and then translates it into a language of the user's
choice.

Building the Sample

Before building and running this sample, you must:

  1. Install the Bing OCR Control and the Bing Translator Control from the Visual
    Studio Gallery
  2. Register both controls with the Azure Data Marketplace to get your ClientIDs
    and ClientSecrets
  3. Add a reference to the Bing.Ocr and Bing.Translator namespaces in your
    project

For more information, see How to: Install and Register the Bing OCR Control with the Azure Data
Marketplace and How to: Register the Bing Translator Control with the Azure Data Marketplace

Description

This sample places an OCR Control and a Translator control on a XAML page,
with two combo boxes to select the languages to translate to and from, two text
blocks to display OCR text and translated text, and a third text block for error
messages. 

In the code behind, it populates to list of languages to read from those
supported by the OCR Control, and populates the list of languages to write from
those supported by thr Translator. It uses the GetLanguageNamesAsync method of
the Translator Control to append the names of the languages to their language
codes.

The OCR control starts from Page Load, and when a user selects an image, the
image goes to the OCR service for processing. The Completed event for the OCR
Control then writes the OCR text to one text box, and then launches the
Translator, which writes the tranlsated text to the other text box.

Source Code Files

  • MainPage.xaml - contains the control declarations and layout of the page.
  • MainPage.xaml.cs - contains the active code for the project.
